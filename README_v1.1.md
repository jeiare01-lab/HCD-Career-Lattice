# HCD Career Lattice v1.1 🎯

**Interactive Career Pathing & Grade Classification Tool for PGB Human Capital Development**

A comprehensive, mobile-friendly React application that visualizes the complete HCD career structure across all job grades (JG1-JG15) and enables employees to build custom career paths with competency gap analysis and integrated PGB Learning & Development training programs.

---

## 🌟 **Key Features**

### **📊 Grade Lattice View**
- Complete visibility of 147+ positions across all HCD functions
- Color-coded by career level (Entry → Intermediate → Officer → Supervisory → Managerial)
- Organized by 10+ functional sections
- Interactive role details with requirements and competencies
- Click any role to see full information

### **🛤️ Interactive Path Builder**
- Build custom career progression paths
- Competency gap analysis between each role transition
- Experience & education requirements clearly displayed
- Real-time transition details showing exactly what's needed to advance

### **📚 PGB L&D Training Integration** ⭐ **NEW IN v1.1**
- 5 training programs mapped to career levels
- 41+ total training modules
- Cost transparency (₱800-₱3,500)
- Training roadmap for multi-step career paths
- Integrated into role details and path builder

### **📤 Export & Sharing**
- 🖨️ **Print** → Professional PDF output with training recommendations
- 💾 **Save as JSON** → Structured data for records
- 📊 **Export as CSV** → Spreadsheet-compatible format
- 📧 **Send to Supervisor** → Formal IDP discussion request form

### **📱 Mobile Optimized**
- Android-first design
- Touch-friendly interface
- Responsive layout
- Full functionality on any device

## 🚀 **Quick Start**

### **Access the App**

**Live URL:**
```
https://hcd-career-lattice.vercel.app
```

No installation needed — just open in any web browser!

### **How to Use**

#### **1. Explore the Grade Lattice**
- Click **"📊 Grade Lattice"** tab
- Browse all 147+ positions in the color-coded grid
- Click any role to see:
  - Grade, level, and functional area
  - Experience & education requirements
  - Core competencies with proficiency levels
  - **Recommended training program for that level**

#### **2. Build Your Career Path**
- Click **"🛤️ Build My Path"** tab
- From role details, click **"+ Add to My Path"**
- Add multiple roles to create your progression
- View between each step:
  - Core competencies to develop
  - Technical competencies to develop
  - Additional experience/education needed

#### **3. See Your Training Roadmap**
- Path builder automatically shows:
  - **Training Roadmap for Your Path** section
  - All training programs needed for your progression
  - Program name, cost, and sample modules
  - Total L&D investment for your path

#### **4. Export or Share**
- **🖨️ Print** → Browser print dialog (saves as PDF)
- **💾 Save JSON** → Download structured data
- **📊 Export CSV** → Download for spreadsheet
- **📧 Send to Supervisor** → Opens IDP discussion form

## 📖 **User Guide**

### **Scenario 1: Career Exploration**
1. Open app → Grade Lattice tab
2. Browse all roles by scrolling grid
3. Click on roles of interest
4. See what's required at each level
5. Understand training needed at each stage

### **Scenario 2: Building a Career Path**
1. Grade Lattice → Click role → "Add to My Path"
2. Repeat for 2-3 more roles to build progression
3. Switch to "Build My Path" tab
4. See gap analysis between each step
5. View training roadmap
6. Print or export for discussion

### **Scenario 3: Supervisor Discussion**
1. Build your desired path
2. Click "📧 Send to Supervisor"
3. Fill form:
   - Supervisor name & email
   - Your career aspirations
   - Preferred discussion date
4. Message generated with full path details
5. Share with supervisor for IDP discussion

## 🛠️ **Customization**

### **Add New Roles**

Edit the `allRoles` array in the component:

```javascript
{
  title: "New Position",
  grade: "JG8",
  level: "Officer",
  function: "TVET",
  section: "Training & Assessment",
  experience: "7-9 years",
  education: "Bachelor"
}
```

### **Change Training Programs**

Edit the `trainingPrograms` object:

```javascript
managerial: {
  program: "Your Program Name",
  cost: "₱X,XXX.00",
  priority: "YOUR AUDIENCE",
  modules: ["Module 1", "Module 2", ...]
}
```

### **Update Colors**

Edit the `colors` object at component start:

```javascript
const colors = {
  navy: '#0D1B3D',
  orange: '#E65100',
  cream: '#FEF9F3',
  // etc.
}
```

## 📊 **Data Reference**

### **Role Object Structure**
```javascript
{
  title: string,           // Position title
  grade: string,           // JG1-JG15
```
