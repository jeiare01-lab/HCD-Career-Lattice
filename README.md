# HCD Career Lattice Live

**Complete Multi-Pillar Career Development Platform for PGB**

---

## 🎯 Overview

HCD Career Lattice Live is a comprehensive career development system covering all four PGB pillars:
- 🛠️ **TVET** - Technical Vocational Education & Training
- 📚 **SHS** - Senior High School  
- 🏢 **Shared Services** - HR, Finance, IT, Operations
- 🤝 **Corporate Foundation** - CSR & Workforce Development

**System Specifications:**
- **5 Career Levels** - SPECIALIST → STRATEGIC PARTNER
- **21 Competencies** - Detailed framework with evidence-based matrix
- **40 Roles** - 10 per pillar, mapped to JG1-JG10
- **Interactive Web App** - Role explorer, self-assessment, career paths
- **Excel Tools** - Pillar-specific competency assessment workbooks
- **Career Guides** - Comprehensive DOCX guides for each pillar

## 🎯 Features

### 🔍 Role Explorer
- Browse 40 roles across all 4 pillars
- Filter by level (SPECIALIST → STRATEGIC PARTNER)
- See JG mapping and role descriptions

### 📊 Competency Matrix
- 21 competencies with detailed framework
- How competencies evolve across 5 levels

### ✅ Self-Assessment
- Rate yourself on 21 competencies (1-5 scale)
- Get instant competency level estimation
- Download Excel tools for detailed assessment

### 🎯 Career Pathways
- 3 progression paths per pillar (12 total)
- Typical timelines and requirements
