# 🔧 Troubleshooting Guide — HCD Career Lattice v1.1

**Solutions to common issues and problems.**

---

## 🌐 **WEBSITE LOADING ISSUES**

### **Problem: Page won't load / Shows blank screen**

**Possible Causes:**
- Internet connection issue
- Browser cache corrupted
- Server temporarily down
- DNS issues

**Solutions (in order):**

1. **Check Internet Connection**
   - Verify WiFi/data is connected
   - Try visiting google.com to test
   - If that fails, reconnect to internet

2. **Clear Browser Cache**
   - **Chrome:** Ctrl+Shift+Delete → Clear browsing data → All time
   - **Safari:** Develop → Clear Caches
   - **Firefox:** Ctrl+Shift+Delete → Everything → Clear Now
   - **Edge:** Ctrl+Shift+Delete → Clear browsing data

3. **Try Different Browser**
   - Chrome, Firefox, Safari, or Edge
   - If one fails, another might work

4. **Hard Refresh**
   - Press: **Ctrl+Shift+R** (Windows)
   - Or: **Cmd+Shift+R** (Mac)

5. **Try Private/Incognito Mode**
   - Opens fresh browser session
   - Sometimes fixes loading issues

6. **Wait and Try Again**
   - Server might be temporarily down
   - Wait 5-10 minutes
   - Try again

**If still failing:**
- Contact support
- Share browser console error (F12)

---

### **Problem: Page loads very slowly**

**Possible Causes:**
- Slow internet connection
- Too many browser tabs open
- Large browser cache
- Server slowness

**Solutions:**

1. **Check Connection Speed**
   - Test at speedtest.net
   - Minimum 1 Mbps needed
   - If slower, move closer to router

2. **Close Unnecessary Tabs**
   - Close other tabs
   - Restart browser
   - Retry

3. **Clear Cache Again**
   - Browser → Settings → Clear browsing data
   - All time → Clear

4. **Disable Browser Extensions**
   - Ad blockers might slow loading
   - Temporarily disable all extensions
   - Retry

5. **Try Different Network**
   - If on WiFi, try mobile hotspot
   - If on mobile hotspot, try WiFi
   - Rules out network-specific issues

## 🎨 **DISPLAY & APPEARANCE ISSUES**

### **Problem: Text is too small / hard to read**

**Solutions:**

1. **Browser Zoom**
   - **Ctrl++** (Windows)
   - **Cmd++** (Mac)
   - Repeat to get desired size
   - Reset with Ctrl+0 or Cmd+0

2. **Display Settings**
   - Settings → Accessibility → Text size
   - Increase system-wide text size

3. **Mobile Landscape Mode**
   - Rotate phone to landscape
   - Usually increases readability

---

### **Problem: Colors look wrong / not displaying**

**Solutions:**

1. **Hard Refresh Browser**
   - Ctrl+Shift+R (Windows)
   - Cmd+Shift+R (Mac)
   - Forces reload with updated styles

2. **Clear Cache**
   - Browser settings → Clear browsing data
   - Reopen app

3. **Check Display Settings**
   - Might be high contrast mode enabled
   - Disable if you want original colors

4. **Try Different Browser**
   - Some browsers handle colors differently
   - Chrome usually most reliable

---

### **Problem: Buttons or text overlapping**

**Solutions:**

1. **Refresh Page**
   - F5 or Cmd+R
   - Rerenders all elements

2. **Adjust Browser Zoom**
   - Ctrl+0 to reset to 100%
   - Then zoom to comfortable size

3. **Use Different Device Orientation**
   - Mobile: Try landscape/portrait
   - Tablet: Rotate to best size

4. **Try Different Browser**
   - Firefox or Safari instead of Chrome
   - Or vice versa

## ⚙️ **FUNCTIONALITY ISSUES**

### **Problem: Buttons not responding / clicking doesn't work**

**Solutions:**

1. **Wait for Page to Fully Load**
   - Give page 3-5 seconds after opening
   - Elements might not be interactive yet

2. **Hard Refresh**
   - Ctrl+Shift+R (Windows)
   - Cmd+Shift+R (Mac)

3. **Close Other Applications**
   - Frees up device memory
   - Might improve responsiveness

4. **Try Double-Click**
   - Single click sometimes requires double on some devices

5. **Mobile: Ensure Touch Works**
   - Try tap instead of hold
   - Ensure finger touches center of button
   - Wait 1 second between taps

6. **Check Browser Console for Errors**
   - Press F12
   - Click "Console" tab
   - Look for red error messages
   - Screenshot and share if asking for help

---

### **Problem: Can't click/select roles in Grade Lattice**

**Solutions:**

1. **Make Sure You're Clicking Actual Buttons**
   - Role names are colored buttons
   - Click on the text/button area

2. **Wait for Grid to Fully Load**
   - Lattice has 147+ roles
   - Takes moment to render
   - Wait 3-5 seconds

3. **Try Single Click**
   - Not double-click
   - One tap/click per role

4. **Scroll to Ensure Role is Visible**
   - Make sure the role you want is fully in view
   - Then click

5. **Try Different Section**
   - Some sections have more roles
   - Try clicking role in different area
   - Verify clicking works elsewhere

---

### **Problem: Detail panel doesn't open when clicking role**

**Solutions:**

1. **Make Sure You're Clicking the Role**
   - Role names are buttons/links
   - Not just text labels
   - Click on the colored box with text

2. **Check Screen Size**
   - On very small screens, panel might be hidden
   - Try landscape mode
   - Or rotate device

3. **Scroll Right to See Panel**
   - On mobile, panel might open off-screen
   - Scroll right to see it
   - Or try landscape orientation

4. **Refresh Page**
   - F5 or Cmd+R
   - Retry clicking

---

### **Problem: "Add to My Path" button grayed out or won't work**

**Solutions:**

1. **Check Button Status**
   - If grayed out with "Added to Path" text
   - That role is already in your path
   - This is normal

2. **To Add Different Role**
   - Click different role first
   - Then click "Add to My Path"
   - That role's button will be available

3. **To Reset**
   - Go to "Build My Path" tab
   - Click "Remove" on all roles
   - Buttons in lattice become active again
   - You can now add new roles

---

### **Problem: Path not showing steps correctly**

**Solutions:**

1. **Refresh "Build My Path" Tab**
   - Click the tab again
   - Or click Grade Lattice, then back to Path

2. **Make Sure Roles Were Actually Added**
   - In lattice, role button should say "Added to Path"
   - If it says "+ Add to My Path", wasn't added yet

3. **Check Role Count**
   - Path shows all added roles in order
   - If missing a role, go back and add it

4. **Clear and Rebuild**
   - Click all "Remove" buttons
   - Start fresh with path building

---

### **Problem: Competency gaps showing as blank**

**Solutions:**

1. **Ensure 2+ Roles in Path**
   - Gaps only show between roles
   - Need at least 2 for any transitions
   - Add more roles

2. **Check Role Grades**
   - Gaps mainly appear when moving between grade levels
   - JG4 → JG6 shows more gaps than adjacent grades
   - This is normal

3. **Refresh Page**
   - F5 or Cmd+R
   - Gaps should recalculate

## 📥 **EXPORT & DOWNLOAD ISSUES**

### **Problem: Export/Download button doesn't work**

**Solutions:**

1. **Check Pop-up Blocker**
   - Browser might be blocking download
   - Allow downloads for this site
   - Then retry export

2. **Wait After Clicking**
   - File takes moment to generate
   - Wait 2-3 seconds before trying again

3. **Try Different Export Format**
   - If JSON doesn't work, try CSV
   - If CSV doesn't work, try JSON
   - One format might work while other fails

4. **Check Downloads Folder**
   - File might have downloaded despite no visible change
   - Look in: Downloads folder
   - Or check browser's download history (Ctrl+J)

5. **Try Different Browser**
   - Chrome, Firefox, Safari, or Edge
   - If one fails, another might work

---

### **Problem: Downloaded file is corrupted or won't open**

**Solutions:**

1. **For PDF (Print Path)**
   - Open with Adobe Reader or browser
   - Make sure it fully downloaded (check file size)
   - Re-download if partially downloaded

2. **For JSON File**
   - Open with text editor (Notepad, VS Code)
   - Not Microsoft Word
   - Should see { } brackets and text

3. **For CSV File**
   - Open with Excel or Google Sheets
   - Not Word
   - Should show table with columns

4. **Re-download Fresh Copy**
   - Rebuild your path
   - Export again
   - Original file might be corrupted

---

### **Problem: Downloaded file location not found**

**Solutions:**

1. **Check Browser's Download Location**
   - Settings → Downloads
   - Shows where files go
   - Usually: `~/Downloads/` or `C:\Users\[Name]\Downloads\`

2. **Search Your Computer**
   - **Windows:** Search for filename
   - **Mac:** Spotlight search (Cmd+Space)
   - **Android:** Files app → Downloads

3. **Check Recent Files**
   - File Manager → Recent
   - Often shows recent downloads

4. **Change Download Location**
   - Browser settings → Downloads
   - Choose specific folder
   - Will remember for future downloads

## 🖨️ **PRINT ISSUES**

### **Problem: Print looks wrong / formatting bad**

**Solutions:**

1. **Adjust Print Settings**
   - Click "Print Path" button
   - In print dialog:
     - Margins → Minimal
     - Paper size → Letter or A4
     - Background → Check "Background colors"

2. **Use Print Preview First**
   - Most browsers show preview
   - Check how it looks before printing
   - Adjust settings if needed

3. **Save as PDF Instead**
   - Printer dropdown → "Save as PDF"
   - More reliable than direct print
   - Can store PDF

4. **Try Different Printer**
   - If available
   - Sometimes printer-specific issues
   - PDF approach usually more universal

---

### **Problem: Print only shows half the page / text cut off**

**Solutions:**

1. **Change to Landscape Orientation**
   - Print dialog → Orientation → Landscape
   - More width for content

2. **Reduce Margins**
   - Print dialog → Margins → Minimal/None
   - Removes white space around edges

3. **Adjust Scale**
   - Print dialog → Scale → 85% or 90%
   - Shrinks everything to fit page

4. **Save as PDF**
   - Usually works better than direct print
   - Can zoom in PDF before printing

---

## 📱 **MOBILE-SPECIFIC ISSUES**

### **Problem: Can't scroll lattice grid on mobile**

**Solutions:**

1. **Use Horizontal Scroll**
   - Swipe left/right with finger
   - Not up/down
   - Should scroll through grades

2. **Try Landscape Mode**
   - Rotate phone to landscape
   - Usually makes grid easier to navigate

3. **Zoom Out**
   - Pinch with two fingers to zoom out
   - See more of grid at once
   - Pinch out to zoom back in

4. **Try Different Section**
   - Some sections smaller, easier to view
   - Verify scrolling works elsewhere

---

### **Problem: Buttons too small to tap on mobile**

**Solutions:**

1. **Zoom In First**
   - Pinch two fingers outward
   - Enlarges buttons
   - Makes tapping easier

2. **Use Landscape Mode**
   - Usually larger buttons
   - More screen space

3. **Tap Center of Button**
   - Not edge
   - Ensure full contact with screen

4. **Wait Between Taps**
   - Give device 1 second between actions
   - Prevents double-tap issues

---

### **Problem: Export files won't download on mobile**

**Solutions:**

1. **Check Download Permissions**
   - Settings → Apps → Browser → Permissions
   - Enable "Storage" or "Files" permission
   - Then retry export

2. **Check Files App**
   - On Android: Open Files app
   - Go to Downloads folder
   - File might be there even if you didn't see notification

3. **Try Different Format**
   - If JSON fails, try CSV
   - If CSV fails, try PDF (Print)
   - Different formats sometimes work better

4. **Try Landscape Mode**
   - Sometimes helps with downloads on mobile

---

### **Problem: Supervisor form hard to fill on mobile**

**Solutions:**

1. **Use Landscape Mode**
   - Larger text input fields
   - Easier to type

2. **Zoom In**
   - Pinch outward to enlarge form
   - Makes text easier to see and type

3. **Use Mobile Keyboard Features**
   - Autocorrect usually helps
   - SwiftKey or Gboard suggestions
   - Use them to type faster

4. **Fill Slowly**
   - One field at a time
   - Don't rush between fields
   - Let keyboard close/open naturally

## 🆘 **GETTING HELP**

### **Before Asking for Help**

1. **Document the Problem**
   - What were you doing?
   - What happened?
   - Screenshot if possible
   - Browser/device type

2. **Check Browser Console**
   - Press F12
   - Click "Console"
   - Look for red error messages
   - Screenshot errors

3. **Try These Steps**
   - Clear cache
   - Hard refresh (Ctrl+Shift+R)
   - Try different browser
   - Restart device
   - Check internet connection

### **If Still Having Issues**

Include when asking for help:
- Browser type and version
- Device (phone/tablet/computer)
- Operating system (Windows/Mac/Android)
- What exactly you were doing
- What happened instead
- Any error messages (screenshot)
- Browser console errors (F12)
